# Decland MODX Revo

Простой компонент, который создаст шаблон и ресурс для него.

Страница представляет из себя лендинг, который содержит:
* Баннер с кнопкой
* Обратную форму с капчей
* Информационный блок
* Карту Яндекс (своя метка + блок с контактами)
* Подвал с логотипом и контактами

Все данные можно менять через админку. Пакет не заденет работу сайта.
Отдельная страница со своими файлами и шаблонами.

**Важно:** После установки, подключите ключи reCAPTCHA у Google и заполните их
в системных настройках.

Данный пакет устанвливает следующие компоненты: (если не установлены)

С modx.com:
* FormIt 
* MIGX 

C modstore.pro:
* AjaxForm
* YandexCoordsTv
